Chromify.js

Yo Dawg, I heard you like to wrap your screenshots in a decorative browser frame

So I created a jQuery plugin that wraps your screenshots in a pure-CSS browser frame so you don't have to mess with photoshop while you wrap your screenshots in a decorative browser frame.

Why use this instead of photoshop?

Let me count the ways.

1. It stretches

Which means it automatically gets shorter or longer depending on it's content. This text is purposefully long so you can see the frame scaling when you resize your browser window. I guess you can call it 'responsive', but seriously folkes, it's just a percentage width. Nothing fancy.

2. It zooms

It uses EM based sizing so all you have to do is
modify the base font-size, and it zooms accordingly.

This frame has a font-size of 10px,
instead of the default 12px.

This is 6px frame
Still lookin' good!

3. Dynamic Content

You can change the tab title and the address-bar title.
Remember when you had to do that in photoshop?
